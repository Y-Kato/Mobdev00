# スネークバードゲーム仕様

## 1. 概要

- タイトル: Snake Bird Mini
- ジャンル: 1画面完結のグリッドアクション
- 目的: フルーツを8個集めてクリアする
- 失敗条件: 壁・岩・自分の体に衝突する

## 2. 画面構成

- ヘッダー領域
  - タイトル表示
  - 説明文表示
- ステータス領域
  - Score 表示（現在値 / 8）
  - 状態メッセージ表示（プレイ中 / 勝利 / 失敗）
- ゲームボード
  - 12 x 12 のセルで構成
  - 要素: 蛇の頭、蛇の体、フルーツ、岩、空セル
- 操作領域
  - モバイル向け方向ボタン（上・左・下・右）
  - リスタートボタン

## 3. 初期状態

- グリッドサイズ: 12
- 勝利スコア: 8
- 初期蛇座標
  - 頭: (2, 6)
  - 体: (1, 6), (0, 6)
- 初期進行方向: 右
- 固定障害物（岩）座標
  - (4, 3)
  - (7, 4)
  - (3, 7)
  - (8, 8)
- フルーツ生成
  - 蛇と岩があるセル以外からランダムに1つ配置

## 4. 操作仕様

- キーボード
  - 上: ArrowUp または W
  - 右: ArrowRight または D
  - 下: ArrowDown または S
  - 左: ArrowLeft または A
- モバイル
  - 画面上の方向ボタンで同等操作
- 入力単位
  - 1回のキー入力またはボタン押下で1手進む
  - キー未入力時は移動しない

## 5. 更新ループ仕様

- 自動更新間隔: なし（ターン制）
- 1手ごとの処理
  1. 入力方向に頭を1セル移動
  2. 衝突判定
  3. フルーツ取得判定
  4. 重力解決（全身を下方向へ、落下不可になるまで移動）
  5. スコア・状態更新

## 6. 判定仕様

- 衝突判定
  - 次の頭座標がグリッド外なら失敗
  - 次の頭座標が岩セルなら失敗
  - 次の頭座標が自分の体セルなら失敗
- フルーツ取得判定
  - 移動後の頭座標がフルーツ座標と一致した場合に取得成功
  - 取得時は蛇が1セル伸びる
  - スコアを1加算
- 勝利判定
  - スコアが8に到達した時点で勝利

## 7. 状態メッセージ

- プレイ中: 矢印キー / WASD で移動。フルーツを集めよう。
- 勝利: ゴール。フルーツを集めきって飛び立ちました。
- 失敗: ぶつかってしまいました。もう一度。

## 8. リスタート仕様

- リスタート押下時
  - 初期蛇配置に戻す
  - 方向を右に戻す
  - スコアを0に戻す
  - 状態をプレイ中に戻す
  - フルーツを再抽選する

## 9. 非機能・表示仕様

- レスポンシブ対応
  - 640px以下では余白やボードギャップを縮小
  - ステータスを縦積み表示
- アクセシビリティ
  - ステータスは aria-live で読み上げ対象
  - ボード・操作領域に aria-label を設定
