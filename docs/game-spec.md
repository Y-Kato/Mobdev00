# スネークバードゲーム仕様

## 1. 概要

- タイトル: Snake Bird Mini
- ジャンル: 固定ステージ型のグリッドパズル
- 目的: 固定配置のフルーツ3個を回収し、ゴールゲートへ到達する
- 失敗条件: 壁・岩・自分の体に衝突する

## 2. 画面構成

- ヘッダー領域
  - タイトル表示
  - 説明文表示
- ステータス領域
  - Fruits 表示（現在値 / 3）
  - Moves 表示（手数）
  - 状態メッセージ表示（プレイ中 / 勝利 / 失敗）
- ゲームボード
  - 12 x 12 のセルで構成
  - 要素: 蛇の頭、蛇の体、フルーツ、岩、ゲート、空セル
- 操作領域
  - モバイル向け方向ボタン（上・左・下・右）
  - リスタートボタン

## 3. 初期状態

- グリッドサイズ: 12
- 初期蛇座標
  - 頭: (2, 7)
  - 体: (1, 7), (0, 7)
- 固定障害物（岩）座標
  - y=11 の x=0..11（床）
  - (0, 8), (1, 8), (2, 8), (3, 8)
  - (6, 8), (7, 8), (8, 8), (9, 8), (10, 8)
  - (2, 5), (3, 5), (4, 5), (5, 5)
  - (8, 4), (9, 4), (10, 4)
- フルーツ固定座標
  - (4, 4), (7, 7), (9, 3)
- ゴールゲート座標
  - (11, 10)

## 4. 操作仕様

- キーボード
  - 上: ArrowUp または W
  - 右: ArrowRight または D
  - 下: ArrowDown または S
  - 左: ArrowLeft または A
- モバイル
  - 画面上の方向ボタンで同等操作
- 入力単位
  - 1回のキー入力またはボタン押下で1手進む
  - キー未入力時は移動しない

## 5. 更新ループ仕様

- 自動更新間隔: なし（ターン制）
- 1手ごとの処理
  1. 入力方向に頭を1セル移動
  2. 衝突判定
  3. 重力解決（全身を下方向へ、落下不可になるまで移動）
  4. フルーツ取得判定
  5. 勝利判定（全フルーツ回収済み かつ 頭がゲート座標）
  6. 状態更新

## 6. 判定仕様

- 衝突判定
  - 次の頭座標がグリッド外なら失敗
  - 次の頭座標が岩セルなら失敗
  - 次の頭座標が自分の体セルなら失敗
- フルーツ取得判定
  - 重力解決後の頭座標がフルーツ座標と一致した場合に取得成功
  - 取得時は蛇が1セル伸びる
  - Fruits カウントを1加算
- 勝利判定
  - フルーツを3個すべて回収後、頭がゲート座標にある場合に勝利

## 7. 状態メッセージ

- プレイ中（フルーツ残あり）: 1手ごとに重力が働きます。先にフルーツを全部回収してください。
- プレイ中（回収完了後）: フルーツ回収完了。右下のゲートへ向かってください。
- 勝利: クリア。全フルーツ回収後にゴールへ到達しました。
- 失敗: ぶつかってしまいました。もう一度。

## 8. リスタート仕様

- リスタート押下時
  - 初期蛇配置に戻す
  - フルーツ配置を初期状態に戻す
  - Fruits / Moves を0に戻す
  - 状態をプレイ中に戻す

## 9. 非機能・表示仕様

- レスポンシブ対応
  - 640px以下では余白やボードギャップを縮小
  - ステータスを縦積み表示
- アクセシビリティ
  - ステータスは aria-live で読み上げ対象
  - ボード・操作領域に aria-label を設定
